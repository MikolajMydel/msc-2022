## Pobranie projektu

```bash
git clone https://github.com/MikolajMydel/msc-2022.git && cd msc-2022
npm install
```

## Uruchomienie serwera deweloperskiego

```bash
npm run dev
```

## Uruchomienie testów automatycznych

```bash
npm test
```
